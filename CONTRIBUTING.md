# Contributing

Thank you for your interest in contributing!

This repository started as a personal learning project while exploring AI tools, GitHub Copilot, Ollama, and open-source LLMs. I decided to make it open source so that other developers can learn, share ideas, and improve the guide together.

## How to Contribute

1. Fork the repository.
2. Create a feature branch from `main`.
3. Make your changes.
4. Commit with a meaningful commit message.
5. Push your branch to your fork.
6. Open a Pull Request.

## Contribution Ideas

* Fix typos or grammatical issues
* Improve explanations
* Add diagrams or examples
* Update model recommendations
* Add new AI tools, models, or workflows
* Improve installation and setup guides

## Pull Request Review Process

All contributions are reviewed before merging.

Please ensure that:

* Documentation is clear and beginner-friendly
* Information is accurate and factually correct
* Changes are relevant to the repository's goal

Thank you for helping improve this project!
